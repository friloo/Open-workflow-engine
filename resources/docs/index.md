# Open Workflow Engine — Anleitung

Willkommen. Die OWE ist eine self-hosted Workflow- und Formular-Plattform.

## In Kuerze

- **Workflows** automatisieren Genehmigungen und wiederkehrende Pruefungen.
- **Formulare** sammeln Daten — intern oder oeffentlich ohne Login.
- **Listen** mappen Schluessel (z. B. Kostenstellen) auf Verantwortliche und Eskalations-Empfaenger.
- **Assets** sind nutzerbezogene Dokumente mit Ablaufdatum (Fuehrerschein, Unterweisung).
- **HTTP-Knoten** integriert externe Systeme (Ticketsysteme, ERP, HR) inkl. **KI-Unterstuetzung** beim Konfigurieren.
- **Webhooks** benachrichtigen externe Systeme bei Workflow-Ereignissen.
- **Audit-Log** ist hashverkettet und revisionssicher.

## Erste Schritte

1. **Systemeinstellungen** ausfuellen (SMTP, optional M365, optional KI).
2. **Listen** anlegen (z. B. Kostenstellen).
3. **Workflow** im Designer entwerfen.
4. **Workflow aktivieren** und ueber Formular starten.
